# Overview

I developed a C# console-based Tic-Tac-Toe game that allows two players to play against each other in a turn-based format. The program includes a menu system, input validation, game state management, and a persistent leaderboard that saves player statistics between program runs using a CSV file.

I wrote this software with the main goal of becoming more familiar with C# syntax and furthering my understanding of object-oriented programming.
